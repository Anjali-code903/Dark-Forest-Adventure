# The Dark Forest - An HTML Adventure

A retro, text-and-image-based "Choose Your Own Adventure" game built entirely from scratch. 

## 🎮 How to Play
You can play the live game right here: [INSERT YOUR GITHUB PAGES LINK HERE]

## 🛠️ Built With
* **100% Pure HTML5** - No CSS, no JavaScript. 
* Uses HTML anchor tags for multi-page navigation.
* Uses native HTML attributes for image rendering.

## 🧠 What I Learned
This was my very first web development project! While building this game, I learned:
1. The foundational structure of an HTML document (`<!DOCTYPE html>`, `<html>`, `<body>`).
2. What `<meta charset="UTF-8">` means and why character encoding matters.
3. How to link multiple separate HTML files together using the `<a>` tag and `href`.
4. How to embed and resize web images using the `<img>` tag.
5. How to deploy a live website globally using **GitHub Pages**.
